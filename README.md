# 🌦️ WeatherTrack - Spring Boot Weather API  

WeatherTrack is a simple Spring Boot application that fetches real-time weather details such as **temperature, wind speed, and weather description** using OpenWeatherMap API. This project follows a clean architecture with RESTful APIs and JSON responses.

---

## 🚀 Features  
✅ Get **current temperature, wind speed, and weather description** for any city  
✅ Uses **Spring Boot & RESTful APIs** for efficient data retrieval  

---

## 📌 Tech Stack  
- **Backend:** Java, Spring Boot, Spring Web  
- **API:** OpenWeatherMap  
- **Data Format:** JSON
